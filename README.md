# 🏎️ Heat: Pedal to the Metal – JavaFX Board Game

## 🚀 Opis projekta

Ovaj projekt je digitalna implementacija društvene igre **Heat: Pedal to the Metal**, razvijena pomoću **JavaFX-a**. Cilj je simulirati dinamičnu utrku uz naglasak na strategiju, upravljanje brzinom i resursima.

## 🛠️ Tehnologije

* Java
* JavaFX
* Objektno orijentirano programiranje (OOP)
  

## ▶️ Pokretanje projekta

### Koraci

1. Kloniraj repozitorij:

```
git clone https://github.com/msaravanj/HeatBoardGame.git
```

2. Otvori projekt u IDE-u (preporuka: IntelliJ)

3. Podesi JavaFX ovisnosti (VM options ili build path)

4. Pokreni:

```
Main.java
```

## 🎯 Gameplay

Igrači upravljaju trkaćim automobilima koristeći kartice koje određuju brzinu i kretanje. Ključno je pravilno upravljati **brzinom** i **heat resursima** kako bi se optimizirala utrka i izbjegla preopterećenja.
